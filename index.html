벤치마크 설명

📚

MMLU-Pro

Massive Multitask Language Understanding의 확장판. 57개 이상의 학문 분야에 걸친 지식과 추론 능력을 평가합니다.

🔬

GPQA Diamond

Graduate-Level Google-Proof Q&A. 대학원 수준의 과학 질문으로, 전문가조차 어려워하는 문제들을 포함합니다.

💻

HumanEval

코드 생성 능력을 평가하는 벤치마크. Python 함수를 올바르게 생성하는 능력을 테스트합니다.

🛠️

SWE-Bench Verified

실제 GitHub 이슈를 해결하는 소프트웨어 엔지니어링 능력 평가. 실무 코딩 능력의 핵심 지표입니다.

🧮

MATH

수학 경시대회 수준의 문제 해결 능력을 평가합니다. 고급 수학적 추론 능력이 필요합니다.

🧠

SimpleBench

상식적 추론 능력을 평가하는 벤치마크. AI의 일반 상식과 논리적 사고력을 테스트합니다.
